**📚 PHỤ LỤC: GIẢI THÍCH THUẬT NGỮ KỸ THUẬT**

|  |  |
| --- | --- |
| **Thuật ngữ** | **Giải thích đơn giản (Dễ hiểu cho khách hàng)** |
| **NVMe U.2** | Là loại ổ cứng đời mới nhất, nhanh hơn ổ SSD thường gấp 10 lần. Giống như đi xe máy (SSD) so với đi máy bay (NVMe). |
| **Băng thông (Bandwidth)** | Là độ lớn của "con đường" truyền dữ liệu. Băng thông Unlimited (không giới hạn) nghĩa là đường rộng thênh thang, bao nhiêu khách vào cũng không bị tắc đường. |
| **CloudLinux** | Là hệ điều hành giúp "chia phòng" cho từng khách hàng trên server. Web nhà này cháy (lỗi/tấn công) thì nhà bên cạnh vẫn an toàn. |
| **LiteSpeed Web Server** | Phần mềm giúp server xử lý yêu cầu nhanh hơn Apache truyền thống. Đặc biệt tốt cho việc chịu tải nhiều người truy cập cùng lúc. |
| **IP Class C** | Địa chỉ IP giúp Google phân biệt các website nằm ở các mạng khác nhau. Rất quan trọng cho SEO để tránh bị Google phạt spam. |
| **Imunify360** | Là "bảo vệ" dùng AI để tự động phát hiện hacker và chặn mã độc ngay lập tức. |
| **MSSQL vs MySQL** | MSSQL là Database của Microsoft (.NET). MySQL là Database mã nguồn mở (PHP). Dùng sai loại web không chạy. |
