# Web Hosting BKNS — So sánh

## Hosting vs Cloud VPS vs Dedicated Server
| Tiêu chí | Hosting | Cloud VPS | Dedicated Server |
|---|---|---|---|
| Chi phí đầu vào | Thấp | Trung bình | Cao |
| Quản trị hệ thống | Dễ | Trung bình đến cao | Cao |
| Quyền kiểm soát | Hạn chế | Cao hơn | Cao nhất |
| Phù hợp | Website doanh nghiệp nhỏ/vừa, blog, landing page | Website tăng trưởng, app riêng, nhiều traffic | Hệ thống tải lớn, yêu cầu tài nguyên vật lý riêng |
| Thời gian triển khai | Nhanh | Nhanh | Chậm hơn |

## So sánh nội bộ trong nhóm hosting
| Nhánh | Dành cho ai | Mức độ dữ liệu đã có |
|---|---|---|
| Platinum Web Hosting | Hub chính / hiệu năng cao | Cao |
| Hosting WordPress | Website WordPress | Trung bình |
| Hosting SEO | Website định hướng SEO | Thấp |
| Hosting Windows | Ứng dụng cần Windows | Thấp |
| Reseller Hosting | Đại lý / agency | Thấp |

## Rule điều hướng bot
- Nếu khách hỏi “website công ty đơn giản” → ưu tiên hosting
- Nếu khách hỏi “cần quyền root / cài phần mềm riêng” → chuyển sang VPS
- Nếu khách hỏi “cần toàn quyền phần cứng” → chuyển sang server
