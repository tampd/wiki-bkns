# Web Hosting BKNS — Tính năng

## USP chung của nhóm hosting
- CPU Platinum / hiệu năng cao
- Ổ NVMe U.2
- Backup định kỳ 1 lần/ngày
- Bảo mật đa tầng với CloudLinux và Imunify360
- LiteSpeed Web Server
- Jetbackup, OneClick, Redis
- Hỗ trợ 24/7
- Dùng thử 7 ngày miễn phí
- Miễn phí chuyển dữ liệu
- Cam kết không tăng giá gia hạn
- Máy chủ tại Việt Nam (theo landing page Platinum)

## Tính năng cần tách ra thành claims riêng
- Backup retention 30 ngày
- Dùng thử 7 ngày
- Không tăng giá gia hạn
- Miễn phí migration
- Hỗ trợ Node.js v6–v18
- PHP 5.6–8.2

## Lưu ý biên tập
- Với nội dung “tăng tốc x10” hoặc “load dưới 1 giây”, nên gắn cờ là marketing claim và lưu cùng source URL
- Các features của Hosting WordPress / SEO / Windows không nên copy từ Platinum nếu chưa có source tương ứng
