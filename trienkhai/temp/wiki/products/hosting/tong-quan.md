# Web Hosting BKNS — Tổng quan chi tiết

## Tổng quan
BKNS đang public nhiều nhánh hosting khác nhau, trong đó Platinum Web Hosting là landing page trung tâm cho phân khúc hosting hiệu năng cao; ngoài ra menu public còn tách riêng Hosting Giá Rẻ, Hosting SEO, Hosting WordPress, Hosting Windows, Reseller Hosting, Hosting Free và NVME Hosting.

## Đối tượng phù hợp
### 1. Doanh nghiệp cần website chạy ổn định
- Ưu tiên giao diện quản trị đơn giản
- Không muốn tự quản trị hệ điều hành / firewall / OS level

### 2. Chủ web mới hoặc đội marketing
- Cần triển khai nhanh
- Muốn gói giá thấp để test MVP
- Muốn dùng sẵn cPanel / WordPress

### 3. Đại lý / agency
- Cần tách trang Reseller Hosting
- Cần bảng so sánh rõ giữa số domain, dung lượng, tài nguyên
