# Chứng Chỉ SSL BKNS — Hướng dẫn

## Quy trình triển khai đề xuất
1. Xác định nhu cầu: DV / OV / EV, single / wildcard / multi-domain
2. Chọn sản phẩm phù hợp
3. Chuẩn bị CSR / domain / thông tin tổ chức
4. Hoàn tất đăng ký và thanh toán
5. Thực hiện xác minh
6. Nhận chứng chỉ
7. Cài đặt lên hosting / VPS / server
8. Kiểm tra HTTPS và gia hạn định kỳ

## Hướng dẫn nên tách ra thành các file nhỏ hơn
- Tạo CSR
- Cài SSL trên Hosting
- Cài SSL trên DirectAdmin / cPanel / Plesk
- Reissue chứng chỉ
- Gia hạn chứng chỉ
- Xử lý lỗi mixed content / SSL chain

## Checklist sau cài đặt
- Kiểm tra HTTPS hoạt động
- Kiểm tra redirect HTTP -> HTTPS
- Kiểm tra expiry date
- Lưu thông tin gia hạn / owner / email xác minh
