# Chứng Chỉ SSL BKNS — Trang tổng quan

> Hub category cho nhóm SSL / Code Signing của BKNS. Category này cần phân theo brand, level xác thực và phạm vi bảo vệ, nên không thể chỉ gói vào một bảng giá phẳng.

## Cụm sản phẩm đang theo dõi
- Positive SSL
- Positive SSL Multi-domain
- Positive SSL Wildcard
- InstantSSL Pro
- Comodo SSL UCC OV
- Comodo Multi-domain SSL
- Comodo EV SSL
- Premium SSL Wildcard
- Comodo EV Multi Domain SSL
- GeoTrust SSL
- DigiCert SSL
- Code Signing

## Cách tổ chức dữ liệu nên dùng trong code
- `brand`: Sectigo / GeoTrust / DigiCert / Code Signing
- `validation_level`: DV / OV / EV
- `scope`: single-domain / multi-domain / wildcard / code-signing
- `insurance`
- `reissue_period`
- `supported_san`
- `green_bar`
