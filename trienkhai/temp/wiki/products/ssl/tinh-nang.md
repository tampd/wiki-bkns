# Chứng Chỉ SSL BKNS — Tính năng

## Tính năng cốt lõi toàn category
- Mã hóa dữ liệu
- Xác thực website / tổ chức
- Tăng độ tin cậy cho website và giao dịch
- Có phân tầng DV / OV / EV
- Hỗ trợ nhiều mô hình: single-domain, multi-domain, wildcard
- Hỗ trợ reissue

## Tính năng nên thể hiện theo từng dòng
| Dòng | Điểm nhấn |
|---|---|
| Positive SSL | DV cơ bản, chi phí thấp |
| Multi-domain | Bảo vệ nhiều domain, có SAN |
| Wildcard | Bao phủ subdomain |
| OV / EV | Mức xác thực cao hơn, độ tin cậy cao hơn |
| Code Signing | Ký ứng dụng, use case khác website |
