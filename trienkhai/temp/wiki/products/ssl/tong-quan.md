# Chứng Chỉ SSL BKNS — Tổng quan chi tiết

## Vai trò của category SSL
SSL không chỉ là “bật khóa xanh” mà còn là nhóm sản phẩm có nhiều biến thể về:
- mức xác thực: DV / OV / EV
- phạm vi bảo vệ: 1 domain / multi-domain / wildcard
- mức độ tin cậy và bảo hiểm
- use case đặc thù: website, doanh nghiệp, phần mềm / code signing

## Phân nhóm sản phẩm nên hiển thị trong wiki
### 1. Sectigo / Comodo
Đây là cụm có landing page bảng giá chi tiết nhất hiện tại, phù hợp làm nguồn public khởi đầu cho schema.

### 2. GeoTrust
Cần page riêng vì BKNS menu và homepage tách brand này như một nhóm độc lập.

### 3. DigiCert
Tương tự GeoTrust, cần page thương hiệu riêng vì định vị cao cấp hơn.

### 4. Code Signing
Không nên trộn với SSL website vì use case, policy và quy trình xác minh khác.

## Đối tượng phù hợp
| Nhu cầu | Gợi ý |
|---|---|
| Website cơ bản cần HTTPS | DV single-domain |
| Nhiều domain / microsite | Multi-domain |
| Toàn bộ subdomain | Wildcard |
| Doanh nghiệp cần mức tin cậy cao hơn | OV / EV |
| Ký ứng dụng / giảm cảnh báo publisher | Code Signing |

## Notes biên soạn
- Tránh gom tất cả SSL vào một bảng duy nhất nếu UI khó đọc
- Nên có bộ lọc theo brand / level / scope
