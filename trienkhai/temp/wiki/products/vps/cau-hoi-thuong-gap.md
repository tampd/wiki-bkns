# Cloud VPS BKNS — Câu hỏi thường gặp

## 1. Khi nào nên chọn Cloud VPS thay vì Hosting?
Khi cần quyền kiểm soát hệ thống cao hơn, tách biệt tài nguyên tốt hơn hoặc cần chạy ứng dụng / tool không phù hợp với shared hosting.

## 2. Cloud VPS của BKNS có điểm mạnh gì?
Các landing page nhấn mạnh 100% NVMe, High Availability, hỗ trợ 24/7/365, khả năng khởi tạo nhanh và control panel quản trị.

## 3. Cloud VPS AMD khác gì Cloud VPS VM?
Cloud VPS AMD nhấn vào CPU AMD EPYC Gen 2, NVMe Raid 10, backup tuần/lần và bộ tính năng quản trị sâu hơn trong landing page.

## 4. Có backup hay không?
Draft hiện cho thấy Cloud VPS có backup định kỳ miễn phí; riêng AMD hiện thể hiện backup tuần/lần. Chính sách retention và restore scope vẫn cần verify.

## 5. Có hỗ trợ hệ điều hành nào?
Trang public thể hiện hỗ trợ hệ điều hành đa dạng; chi tiết distro / version cần compile riêng theo claims.

## 6. Tôi có thể nâng tài nguyên sau này không?
Kiến trúc Cloud VPS phù hợp cho scale, nhưng chính sách nâng / hạ cấp và downtime tương ứng cần xác minh theo product claim.
