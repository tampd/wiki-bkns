# Cloud VPS BKNS — Hướng dẫn

## Quy trình đăng ký đề xuất
1. Xác định nhu cầu: VM / AMD / dòng chuyên biệt
2. Chọn gói phù hợp theo CPU, RAM, SSD và network
3. Đăng ký tại `my.bkns.net`
4. Thanh toán hóa đơn
5. Chờ hệ thống khởi tạo
6. Truy cập control panel / console để cài hệ điều hành và cấu hình ban đầu

## Checklist sau khi kích hoạt
- Đổi mật khẩu mặc định
- Kiểm tra network / firewall
- Cấu hình backup
- Ghi lại IP / thông tin truy cập
- Bật giám sát / cảnh báo

## Hướng dẫn nên tách nhỏ thêm về sau
- Hướng dẫn cài Linux
- Hướng dẫn cài Windows
- Hướng dẫn dùng Console / noVNC
- Hướng dẫn dùng Snapshot / Backup
- Hướng dẫn mở firewall / port
- Hướng dẫn recovery mode
