# Cloud VPS BKNS — So sánh

## Cloud VPS VM vs Cloud VPS AMD
| Tiêu chí | Cloud VPS VM | Cloud VPS AMD |
|---|---|---|
| Mục tiêu | Dòng phổ thông / rộng use case | Dòng hiệu năng cao hơn |
| Giá khởi điểm | 119.000đ/tháng | 140.000đ/tháng |
| CPU | Intel Xeon / nền tảng cloud tiêu chuẩn | AMD EPYC Gen 2 |
| Backup | Backup định kỳ miễn phí | Backup tuần/lần |
| USP | HA, NVMe, scale nhanh | AMD EPYC, Infinity Guard, Raid 10 |

## Hosting vs Cloud VPS
| Tiêu chí | Hosting | Cloud VPS |
|---|---|---|
| Panel dựng web | Có sẵn | Có thể cần tự cấu hình |
| Toàn quyền hệ thống | Hạn chế | Có |
| Dễ dùng | Cao | Trung bình |
| Scale tài nguyên | Theo gói | Linh hoạt hơn |

## Cloud VPS vs Server
| Tiêu chí | Cloud VPS | Server |
|---|---|---|
| Khởi tạo | Nhanh | Phụ thuộc phần cứng / triển khai |
| HA | Có thể có ở cloud | Tùy kiến trúc |
| Kiểm soát phần cứng | Không trực tiếp | Cao hơn |
| Phù hợp | SMB đến enterprise vừa | Enterprise, tải rất cao, nhu cầu đặc thù |
