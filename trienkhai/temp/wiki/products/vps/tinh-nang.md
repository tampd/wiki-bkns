# Cloud VPS BKNS — Tính năng

## Tính năng nổi bật toàn category
- 100% ổ cứng NVMe / Enterprise NVMe
- High Availability giúp tăng tính liên tục dịch vụ
- Tự động / nhanh chóng khởi tạo dịch vụ
- Hỗ trợ kỹ thuật 24/7/365
- Hệ điều hành linh hoạt
- Bảng điều khiển dịch vụ với thao tác Reboot / Reinstall / Firewall / Console / Snapshot

## Tính năng nổi bật riêng cho Cloud VPS AMD
- CPU AMD EPYC Gen 2
- Bảo mật với AMD Infinity Guard
- NVMe Raid 10
- Backup tuần/lần
- Hỗ trợ băng thông lớn
- Recovery Mode để khôi phục mật khẩu, kiểm tra network và repair OS

## Tính năng nên tách thành component UI
### 1. Hạ tầng
- NVMe
- HA
- CPU class

### 2. Quản trị
- noVNC Console
- Reinstall OS
- Firewall
- Snapshot
- Upload ISO

### 3. Vận hành
- Backup
- Hỗ trợ kỹ thuật
- Recovery Mode

## Notes biên soạn
- Không gắn “Unlimited” cho mọi dòng nếu chưa có claim cụ thể
- Tính năng phải gắn scope `category`, `family` hoặc `product` để tránh hiển thị sai sang sản phẩm con khác
