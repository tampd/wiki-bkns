# Cloud VPS BKNS — Tổng quan chi tiết

## Cloud VPS VM
Cloud VPS VM là nhóm VPS phổ thông của BKNS, nhấn vào hạ tầng 100% NVMe, High Availability và tốc độ khởi tạo nhanh. Đây là nhánh phù hợp cho web/app đang tăng trưởng, môi trường dev/staging, workload nội bộ cần chi phí hợp lý.

## Cloud VPS AMD
Cloud VPS AMD là nhánh VPS hiệu năng cao dùng CPU AMD EPYC Gen 2, có backup tuần/lần và control panel quản trị sâu. Nhánh này phù hợp cho doanh nghiệp cần hiệu năng tốt hơn, workload xử lý nhiều, agent AI, automation và hệ thống cần băng thông cao.

## VPS Giá Rẻ
Mục tiêu là cost-efficient cho bài toán triển khai nhanh, ngân sách tối ưu. Trong wiki, nhóm này nên có trang riêng để tránh trộn giá thấp với các dòng cao cấp hơn.

## Storage VPS
Nhánh thiên về lưu trữ dung lượng lớn, phục vụ backup, media, archive hoặc file service.

## VPS N8N AI / OpenClaw AI
Đây là nhóm sản phẩm “solution-oriented”, không chỉ bán tài nguyên mà còn bán use case. Vì vậy product page nên có thêm các block:
- đối tượng sử dụng
- workflow mẫu
- dependency / stack đi kèm
- giới hạn hỗ trợ

## Cloud VPS SEO / VPS MMO / Cloud VPS BK Misa
Đây là các biến thể theo bài toán nghiệp vụ cụ thể. Cần page riêng để mô tả:
- IP / network / region (nếu có claims)
- app compatibility
- support boundary
- policy riêng nếu khác dòng VPS chuẩn

## Gợi ý nhóm đối tượng khách hàng
| Nhu cầu | Dòng VPS gợi ý |
|---|---|
| Web/app SMB cần triển khai nhanh | Cloud VPS VM |
| Workload hiệu năng tốt hơn | Cloud VPS AMD |
| Automation / workflow / agent | VPS N8N AI, OpenClaw AI |
| SEO / MMO / MISA | Dòng chuyên biệt tương ứng |
| Lưu trữ / backup | Storage VPS |
