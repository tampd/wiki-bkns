# Bản quyền DirectAdmin

**DirectAdmin** là phần mềm quản trị server (control panel) có giao diện đồ họa, được biết đến như một giải pháp thay thế hiệu quả và tiết kiệm chi phí hơn cho cPanel. BKNS cung cấp bản quyền DirectAdmin chính thức.

## Tổng Quan

- **Loại sản phẩm:** Bản quyền phần mềm (Control Panel)
- **Nền tảng:** Linux (CentOS, CloudLinux, AlmaLinux, Rocky Linux, Ubuntu, Debian)
- **Đối tượng:** Hosting Provider, VPS Owner, Dedicated Server Owner

## Tính Năng Chính

- Giao diện quản lý hosting đơn giản, nhẹ, tốc độ cao
- 3 cấp độ người dùng: Admin / Reseller / User
- Quản lý email, FTP, database, domain, SSL
- Hỗ trợ Multi-PHP version
- Tích hợp tốt với CloudLinux và Imunify360
- Tiêu thụ tài nguyên server thấp hơn cPanel

## Bảng Giá

| Gói | Giá |
|:----|:----|
| DirectAdmin Lite (External License) | 450.000 VNĐ/tháng |
| DirectAdmin Standard (External License) | 790.000 VNĐ/tháng |

> **Lưu ý:** "External License" là giấy phép áp dụng cho server/VPS không nằm trong hạ tầng BKNS.

## Yêu Cầu Hệ Thống

- RAM tối thiểu: 512 MB (khuyến nghị ≥ 1 GB)
- Ổ cứng trống: ≥ 2 GB
- OS: CentOS 7+, AlmaLinux 8/9, Rocky Linux 8/9, Ubuntu 20.04+, Debian 10+

## So Sánh với cPanel

| Tiêu chí | DirectAdmin | cPanel/WHM |
|:---------|:-----------|:---------|
| Giá | Rẻ hơn | Đắt hơn |
| Tài nguyên tiêu thụ | Ít | Nhiều hơn |
| Giao diện | Đơn giản | Nhiều tính năng hơn |
| Độ phổ biến | Vừa | Phổ biến nhất |
