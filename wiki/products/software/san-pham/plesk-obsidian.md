# Bản quyền Plesk Obsidian

**Plesk Obsidian** là phần mềm quản trị hosting cao cấp, hỗ trợ cả Linux và Windows, được sử dụng rộng rãi trong môi trường VPS và máy chủ riêng. BKNS cung cấp bản quyền Plesk chính thức cho nhiều phiên bản.

## Tổng Quan

- **Loại sản phẩm:** Bản quyền phần mềm (Control Panel)
- **Nền tảng:** Linux và Windows Server (duy nhất hỗ trợ cả hai)
- **Đối tượng:** Hosting Provider, Developer, Doanh nghiệp cần hosting Windows

## Tính Năng Chính

- Giao diện hiện đại, thân thiện người dùng
- Hỗ trợ WordPress Toolkit tích hợp sẵn
- Quản lý email, FTP, database, domain, SSL
- Hỗ trợ Docker, Git, Node.js, Python, Ruby
- Panel quản trị tập trung cho nhiều server (Plesk Multi-Server)
- Hỗ trợ Windows IIS và ASP.NET
