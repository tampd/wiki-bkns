## 3. Phần mềm Máy chủ Web

### LiteSpeed Web Server (LSWS)

LiteSpeed Web Server là giải pháp máy chủ web hiệu suất cao với các đặc điểm nổi bật:
*   **Hỗ trợ giao thức:** Hỗ trợ HTTP/3 và QUIC.
*   **Hỗ trợ Rewrite:** Cung cấp cơ chế rewrite hiệu quả hơn so với `Mod_Rewrite` của Apache do được viết lại hoàn toàn.
