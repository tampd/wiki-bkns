# GeoTrust True BusinessID Wildcard

**GeoTrust True BusinessID Wildcard** là chứng chỉ SSL do **GeoTrust (DigiCert)** cung cấp, phân phối chính thức bởi BKNS. Chứng chỉ SSL OV kết hợp Wildcard, bảo mật domain chính và toàn bộ subdomain cấp 1, có xác thực tổ chức.

## Thông Tin Sản Phẩm

| Thuộc tính | Chi tiết |
| :--- | :--- |
| **Nhà cung cấp** | GeoTrust (DigiCert) |
| **Loại xác thực** | Organization Validation (OV) |
| **Mục tiêu bảo vệ** | 1 domain + *.domain (OV) |
