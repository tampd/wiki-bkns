# GeoTrust True BusinessID

**GeoTrust True BusinessID** là chứng chỉ SSL do **GeoTrust (DigiCert)** cung cấp, phân phối chính thức bởi BKNS. Chứng chỉ SSL xác thực doanh nghiệp (OV), xác thực danh tính và tăng độ tin cậy cho website doanh nghiệp.

## Thông Tin Sản Phẩm

| Thuộc tính | Chi tiết |
| :--- | :--- |
| **Nhà cung cấp** | GeoTrust (DigiCert) |
| **Loại xác thực** | Organization Validation (OV) |
| **Mục tiêu bảo vệ** | 1 domain (OV) |
