# Cloud VPS SSD — VM Series

**Cloud VPS SSD** (VM Series) là dịch vụ máy chủ ảo phổ thông của BKNS, sử dụng nền tảng ảo hóa **KVM** và ổ cứng **SSD** tốc độ cao. Phù hợp cho cá nhân, startup và doanh nghiệp cần VPS độc lập, ổn định để chạy website, ứng dụng hoặc lưu trữ dữ liệu với chi phí hợp lý.

## Đặc Điểm Chung

| Thuộc tính | Giá trị |
| :--- | :--- |
| **Ảo hóa** | KVM |
| **Lưu trữ** | SSD |
| **Nền tảng** | Ceph Distributed Storage |
| **Băng thông** | Không giới hạn |
| **Sao lưu** | 1 lần / tuần |
| **Kích hoạt** | Tự động |
| **Quản lý** | Bảng điều khiển VPS |
| **Giám sát** | BKNS Cloud Monitoring |
| **Mạng** | Dual Uplink – Multihome |
| **IPv6** | Có hỗ trợ |
